# LiDAR ガラス検知シミュレーター (LiDAR Glass Detection Simulator)

このアプリケーションは、自動運転やロボットナビゲーションにおける「LiDAR（ライダー）センサによる透明体・屈曲壁の反射特性」を検証・可視化するための高精度なブラウザ・ベースのシミュレーターです。

実際のロボット実験を実施する前に、机上で「どのような条件でLiDARから強い反射強度が得られるか」「角度や距離が変化した際、ガラスのピークがどう推移するか」という光学的・幾何学的なモデルをリアルタイムでシミュレーションすることが可能です。

---

## 🚀 アプリケーションで「できること」

1. **2D鳥瞰図でのLiDARスキャン可視化**
   - LiDAR搭載車両を操作し（`WASD` または矢印キーで移動、`Q` `E` で回転）、前方の点群スキャン範囲（正面を中心とした180度の範囲）を視覚的に表現します。
   - 壁に衝突した点・反射強度・素材の色がマップ上にリアルタイムで表示され、デモ環境としての直感的な操作が可能です。

2. **詳細な壁面エディタ機能**
   - **材質の変更**: ガラス、アクリル、コンクリート、金属など、材質ごとの反射パラメーターを細かくシミュレーション。
   - **形状と配置の変更**: 平面壁・屈曲壁・曲面壁を自由に定義可能。壁の長さ、配置角度はもちろん、屈曲角や曲率半径もUIから動的に変更できます。

3. **反射強度のリアルタイム・データ解析（グラフ化）**
   - **反射強度ヒートマップ**: `[水平角度 × 垂直角度]` の二次元平面上に、得られた反射強度をヒートマップとして展開します。上下角や水平方向への広がりの差（ガウス分布）が視覚的にわかります。
   - **反射強度 3Dグラフ**: `x:水平角度`, `z:垂直角度`, `y:反射強度` とした3D散布図を描画。マウスのドラッグによる回転や、XYZ軸それぞれの1度単位での微調整ボタンを備え、ピークの立ち上がり（鋭さ）を立体的に観察できます。

## 🧠 シミュレーションの仕組みと光学ロジック（教授への説明用）

本シミュレーターは、単なる2Dの当たり判定ゲームではなく、3D空間における光学的な反射モデル（ガウス関数の積）と幾何学的な制約（高さ制限や距離減衰）を実装しています。

### 1. レイキャスティング（Raycasting）によるヒット判定
LiDARの水平スキャン（水平方向の解像度）と垂直スキャン（仰角・俯角方向の解像度）を疑似的に再現しています。
- 各レイについて、空間上の壁の線分との交差判定（Line-Segment Intersection）を行い、ヒットした座標、反射面までの**距離 ($d$)**、および壁の**法線ベクトル**を取得します。
- 水平だけでなく、「LiDARの設置高さ（0.3m）」と「壁の高さ（0〜2.5m）」を使用した三次元的な高さ制限ロジックを保有しており、レイの垂直角度（$\theta_V$）によってビームが壁の上を通過してしまうケースなども正確に処理しています。

### 2. 水平・垂直独立の二変量ガウス分布モデル（反射強度計算）
ガラスのような鏡面反射（Specular Reflection）を起こす透明体は、レーザーの入射角が垂直（0度）の時に最も強い反射（ピーク）を返し、角度がずれると急激に強度が低下します。
これを表現するため、本シミュレータでは**水平方向と垂直方向のレーザー照射の不均一性**をモデル化した「独立した2つのガウス分布の積」を計算しています。

強度の計算式は以下の通りです：

$$ I(\theta_H, \theta_V, d) = I_{max} \cdot \exp\left(-\frac{\theta_H^2}{2\sigma_H^2}\right) \cdot \exp\left(-\frac{\theta_V^2}{2\sigma_V^2}\right) \cdot f_{atten}(d) $$

- **$\theta_H$ (水平入射角)**: レーザー光と壁面法線の水平面上でのなす角（内積から算出）。
- **$\theta_V$ (垂直入射角)**: レーザー光の俯仰角（壁が垂直に建っていると仮定すると、そのまま垂直入射角に相当）。
- **$\sigma_H, \sigma_V$ (分散 / 広がり)**: 材質ごとの反射光の広がりやすさ。ガラス(8)は鋭く、コンクリート(90)は極めて広い乱反射となるよう設定。
- このロジックにより、LiDARレーザーのビーム特性（横長・縦長）による反射特性の違いまでシミュレーション・分析対象に含めることが可能になっています。

### 3. 距離減衰モデル（Distance Attenuation）
LiDARから照射された光のエネルギーは、物体に到達して反射して戻ってくる過程で、光の拡散（逆二乗則）に従って減衰します。
これらを近似するため、基準距離 ($d_{ref}$) を設けた減衰関数 $f_{atten}(d)$ を乗算しています。

$$ f_{atten}(d) = \frac{d_{ref}^2}{d_{ref}^2 + d^2} $$

これにより、ガラスに対して垂直に入射した場合であっても、LiDARからの距離が長くなるに伴って滑らかに反射ピークが下がっていく（実際の機器に近い）振る舞いを再現しました。

## 🔧 各材質のパラメータ設定例 (Physics.MATERIALS)
- **ガラス (Glass)**: 光の大半が透過するため基礎反射率が低く、入射角が0度に近いときのみ強いピークが発生します。（$\sigma_H = 8, \sigma_V = 8$）
- **コンクリート (Concrete)**: ランバート反射（乱反射）に近く、入射角が大きくてもある程度の強度を保って広範囲に光を返します。（$\sigma_H = 90, \sigma_V = 90$）

## 💻 起動・操作方法

1. **シミュレーターの起動**
   - GitHub PagesのURL（公開済みの場合はそこから）を開くか、HTMLファイル (`index.html`) をブラウザで直接開いてください。面倒な環境構築やインストールは一切不要です。（※ローカルで動作拡張を行う場合はLive Server等を利用してください）

2. **LiDAR（車両）の移動と回転**
   - **移動**: 左上の「2D 鳥瞰図」内でクリックしてフォーカスを合わせた後、キーボードの `W, A, S, D` または `↑, ↓, ←, →` キーで車両を前後左右に移動させることができます。
   - **回転**: キーボードの `A, D`（または左右矢印キー）で車体の向きを変えられます。少し細かく旋回したい場合は `Q`（左旋回）と `E`（右旋回）キーを使用してください。

3. **壁（オブジェクト）の操作・配置**
   - **種類・材質の変更**: 左下の「壁エディタ」パネルから、壁の種類（平面・屈曲・曲面）や材質（ガラス・アクリル・金属・コンクリート）を選択します。
   - **パラメータの調整**: スライダーを使用して、壁の「長さ」「角度」などを自由に変更できます。曲面や屈曲壁を選んだ場合は「曲率半径」などの追加パラメータが表示されます。
   - **直感的なマウス操作**: 2Dマップ上に配置された壁は、**マウスのドラッグ＆ドロップで直接つかんで移動（平行移動）**させることができます。また、エディタの一覧から角度の微調整（◀ / ▶ボタン）や削除（✕ボタン）も可能です。
   - **壁の追加**: 「壁を追加」ボタンを押すと、現在の設定に基づいた新しい壁がマップ中央付近にドロップされます。

4. **反射強度 3Dグラフのアングル変更・視点操作**
   - **マウスによる自在な回転**: 3Dグラフの領域内でマウスを**左クリックしたままドラッグ**すると、グラフ全体をぐるぐると立体的に回転・俯瞰させることができます。（ドラッグ操作中はシミュレーションの更新にかき消されることなく、ゆっくりと視点を調整できます）
   - **ズーム・パン**: マウスのホイールスクロール（または2本指スクロール）でグラフの拡大・縮小が行えます。右クリックしながらドラッグで平行移動です。
   - **1度単位の微調整**: グラフ右上のパネルタイトルにある **「X軸: - +」「Y軸: - +」「Z軸: - +」ボタン** をクリックすると、各軸を基準にして1度単位でカメラの視点を完璧に微調整できます。プレゼン等で特定のアングルを固定して見せたい場合に非常に便利です。
